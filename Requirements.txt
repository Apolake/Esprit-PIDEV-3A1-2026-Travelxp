2.2 Gamification Engine

Purpose: Track user activity and assign experience points (XP), levels, and titles.

Entities in Database:

gamification table
CREATE TABLE gamification (
    user_id INT PRIMARY KEY,
    xp INT DEFAULT 0,
    level INT DEFAULT 1,
    title VARCHAR(50) DEFAULT 'Novice',
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);


Rules & Features:

Experience Points (XP)

Earned by user actions:

Login: +10 XP

Complete profile fields: +5 XP each (bio, birthday, image)

Change password/email: +5 XP

Levels

Level thresholds (example):

Level 1: 0 XP

Level 2: 50 XP

Level 3: 120 XP

Level 4: 200 XP

Level 5+: +100 XP per level

Level calculated dynamically based on XP

Titles

Assign a title based on level:

Level 1–2: Novice

Level 3–4: Explorer

Level 5–6: Traveler

Level 7+: Adventurer

Service Responsibilities (GamificationService.java)

addXp(int userId, int points)

calculateLevel(int xp) → int

getTitle(int level) → String

updateGamification(User user, int xpGained)

getGamificationByUserId(int userId) → returns XP, Level, Title

Direct JDBC Usage:

No DAO folder; all database interaction in GamificationService

Use PreparedStatement, handle exceptions, maintain security
